# Документация ИС
*Проект:* ИС записи пациентов на прием (MVP)  
*Формат:* шаблон для проектной документации  
*Технологическая реализация:* Spring Boot + Kotlin + Exposed + PostgreSQL + Redis + Docker + Flyway

---

## Назначение ИС
Информационная система предназначена для ведения справочников (пациенты, врачи, расписания) и 
управления процессом записи на прием: создание записи, подтверждение/отмена, контроль доступности слотов, хранение 
истории и отправка уведомлений.
